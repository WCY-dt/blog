正好下学期要学编译原理，我就提前自己搜罗各种资料，来实现一个 C 编译器。为什么是 C 语言？因为 C 是高级语言，但工作量又不会那么多。实现这玩意儿有啥用？没啥卵用。

当然，编程的最大乐趣就是能看到程序跑起来。我们希望做的是，搞一个更接近于解释器的东西，输入程序能够跑出结果，而不是仅仅生成汇编代码。这其实很简单，我们只需要写一个能生成汇编代码的玩意儿，再写一个能运行汇编代码的玩意儿，咔擦一下给合并起来，不就好了嘛！

上面这么做还有个好处，就是写出来的东西能实现自举。想当年它从打孔纸带一路升级到 C，~~我们从 C 升级到 C++ 也不是梦~~！

阅读本文前，你应当已经学过计算机组成（也叫计算机体系结构）、操作系统、简单的汇编基础。

# 汇编模拟运行

我们先从最简单的汇编模拟运行开始。

首先给出一个例子

```c
int callee(int, int, int);
int caller(void)
{
	int i, ret;
	ret = callee(1, 2, 3);
	ret += 5;
	return ret;
}
```

上面的 C 代码会生成汇编如下

```assembly
caller:
	; make new call frame
	push    ebp
	mov     ebp, esp
    sub     1, esp       ; save stack for variable: i
	; push call arguments
	push    3
	push    2
	push    1
	; call subroutine 'callee'
	call    callee
	; remove arguments from frame
	add     esp, 12
	; use subroutine result
	add     eax, 5
	; restore old call frame
    mov     esp, ebp
	pop     ebp
	; return
	ret
```

我们希望程序能够运行上面的汇编代码——或者至少是近似的代码。

## 寄存器

计算机中除了这样的内存外，还有许多寄存器。其中，我们要用到四个：

- `pc` 程序计数器，存放下一条要执行的指令的地址
- `sp` 指针寄存器，指向当前的栈顶
- `bp` 基址指针
- `ax` 通用寄存器，在这里我们用来存放一条指令执行后的结果

我们先定义这些寄存器

```c
int *pc, *bp, *sp, ax;
```

然后进行初始化

```c
bp = sp = (int *)((int)stack + poolsize);
ax = 0;
```

## 汇编指令集

我们希望能运行汇编语言，那必然要能够处理各种汇编命令。我们列举出所有汇编命令

```c
enum { LEA, IMM, JMP, CALL,JZ , JNZ, ENT, ADJ, LEV, LI , LC , SI , SC , PUSH, OR , XOR, AND, EQ , NE , LT , GT , LE , GE , SHL, SHR, ADD, SUB, MUL, DIV, MOD,  OPEN, READ, CLOS, PRTF, MALC, MSET, MCMP, EXIT };
```

我们定义当前要处理的命令为 `op`

```c
int op;
```

上面的指令中，有一些是非常简单的，一行程序就可以实现

- `IMM <num>` 将立即数 `<num>` 放入寄存器 `ax` 中
- `LC` 将对应地址中的**字符**载入 `ax` 中，要求 `ax` 中存放地址
- `LI` 将对应地址中的**整数**载入 `ax` 中，要求 `ax` 中存放地址
- `SC` 将 `ax` 中的数据作为**字符**存放入地址中，要求栈顶存放地址
- `SI` 将 `ax` 中的数据作为**整数**存放入地址中，要求栈顶存放地址
- `PUSH` 将 `ax` 的值放入栈中
- `JMP <addr>` 将当前的 `PC` 寄存器设置为 `<addr>`
- `JZ` 和 `JNZ` 分别是 `ax` 为零或不为零情况下的跳转
-  `CALL <addr>` 跳转到地址为 `<addr>` 的子函数

实现代码

```c
if (op == IMM) {ax = *pc++;} // 栈顶元素出栈，并将值赋给 ax
else if (op == LC) {ax = *(char *)ax;}
else if (op == LI) {ax = *(int *)ax;}
else if (op == SC) {ax = *(char *)*sp++ = ax;}
else if (op == SI) {*(int *)*sp++ = ax;}
else if (op == PUSH) {*--sp = ax;}
else if (op == JMP)  {pc = (int *)*pc;}
else if (op == JZ)   {pc = ax ? pc + 1 : (int *)*pc;}
else if (op == JNZ)  {pc = ax ? (int *)*pc : pc + 1;}
else if (op == CALL) {*--sp = (int)(pc+1); pc = (int *)*pc;}
```

---

前面我们提到过，计算机有非常多的寄存器，但我们只用了 4 个。这可以简化我们编译器的实现，但对于汇编运行器就不太友好了。为了弥补这一缺陷，我们增加几个现实机器中不存在的指令。

`ENT <size>` 可以保存当前的栈指针，同时在栈上保留一定的空间，用以存放局部变量。它对应汇编代码

```assembly
; make new call frame
push    ebp
mov     ebp, esp
sub     1, esp       ; save stack for variable: i
```

根据汇编，给出它的实现

```c
else if (op == ENT)  {*--sp = (int)bp; bp = sp; sp = sp - *pc++;}
```

`ADJ <size>` 用来替代 `ADD`，将调用子函数时压入栈中的数据清除。它对应汇编代码

```assembly
; remove arguments from frame
add     esp, 12
```

根据汇编，给出它的实现

```c
else if (op == ADJ)  {sp = sp + *pc++;}
```

`LEV` 用来替代 `POP`，实现 `RET` 的作用。它对应汇编代码

```assembly
; restore old call frame
mov     esp, ebp
pop     ebp
; return
ret
```

根据汇编，给出它的实现

```c
else if (op == LEV)  {sp = bp; bp = (int *)*sp++; pc = (int *)*sp++;}
```

想要获取到第一个参数，我们必须要 bp+4，但没有 `ADD` 指令是无法完成的。因此我们提供一个新的指令 `LEA`，其实现如下

```c
else if (op == LEA)  {ax = (int)(bp + *pc++);}
```

---

此外，我们还需要各种运算符。它们先将一个操作数存在 ax 中，然后和另一个操作数运算后把结果存入 ax。具体代码如下

```c
else if (op == OR)  ax = *sp++ | ax;
else if (op == XOR) ax = *sp++ ^ ax;
else if (op == AND) ax = *sp++ & ax;
else if (op == EQ)  ax = *sp++ == ax;
else if (op == NE)  ax = *sp++ != ax;
else if (op == LT)  ax = *sp++ < ax;
else if (op == LE)  ax = *sp++ <= ax;
else if (op == GT)  ax = *sp++ >  ax;
else if (op == GE)  ax = *sp++ >= ax;
else if (op == SHL) ax = *sp++ << ax;
else if (op == SHR) ax = *sp++ >> ax;
else if (op == ADD) ax = *sp++ + ax;
else if (op == SUB) ax = *sp++ - ax;
else if (op == MUL) ax = *sp++ * ax;
else if (op == DIV) ax = *sp++ / ax;
else if (op == MOD) ax = *sp++ % ax;
```

---

对于诸如输入输出函数等，我们可以使用系统调用，以从底层实现它们。但这里我们简化起见，直接使用 libc 已经提供的函数 `exit`、`open`、`close`、`read`、`printf`、`malloc`、`memset` 和 `memcmp`

```c
else if (op == EXIT) { printf("exit(%d)", *sp); return *sp;}
else if (op == OPEN) { ax = open((char *)sp[1], sp[0]); }
else if (op == CLOS) { ax = close(*sp);}
else if (op == READ) { ax = read(sp[2], (char *)sp[1], *sp); }
else if (op == PRTF) { int *tmp; tmp = sp + pc[1]; ax = printf((char *)tmp[-1], tmp[-2], tmp[-3], tmp[-4], tmp[-5], tmp[-6]); }
else if (op == MALC) { ax = (int)malloc(*sp);}
else if (op == MSET) { ax = (int)memset((char *)sp[2], sp[1], *sp);}
else if (op == MCMP) { ax = memcmp((char *)sp[2], (char *)sp[1], *sp);}
```
